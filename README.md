# Fuel App

A mobile-first **Ionic + Vue 3** application for fuel tracking and location-based services.  
This project integrates with **Firebase** and **Google Maps API** for real-time data and map rendering.

Before you begin, make sure you have **Node.js (v18+)**, **npm or yarn**, and the **Ionic CLI** installed globally using:

```bash
npm install -g @ionic/cli
```

Install Dependencies:
```bash
cd frontend && npm i
```

Run the application:
```bash
npm run dev
```

Open your browser and visit: http://localhost:8100/
