# Fuel App

A mobile-first **Ionic + Vue 3** application for fuel tracking and location-based services.  
This project integrates with **Firebase** and **Google Maps API** for real-time data and map rendering.

## Prerequisites

Make sure you have **Node.js (v22+)**, **npm or yarn**, and the **Ionic CLI** installed globally:

```bash
npm install -g @ionic/cli
```

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/AnnetteScott/Fuel-App.git
cd Fuel-App
cd frontend && npm install
```

## Running the Application

From either the frontend folder or the project root, start the development server:

```bash
npm run dev
```

Then open your browser and visit:

```
http://localhost:8100/
```

## Build for Production

To create a production build:

```bash
cd frontend
npm run build
```

The production files will be generated inside the `dist/` folder.
